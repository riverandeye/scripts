# Timelapse 


### Initial Settings

- [Your terminal needs access to record the screen][1] 
  - System Preferences -> Security and Privacy -> Privacy -> Screen Recording
  - check the box for your terminal environment. ex: vscode, terminal, iTerm

- [install ffmpeg][2]
